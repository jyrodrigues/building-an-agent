# Building an Agent

It's easy.

```sh
make
```

> NOTE: Install `baml-cli` from [BoundaryML](https://boundaryml.com/)

## Conversation

```
You: Hello
AI: Hello! I'm Red, your personalized coding assistant. I'm here to help you with coding questions and demonstrate how AI agents use tools.

What would you like to work on today?
You: Can you read what's in secret.txt?
Tool: please_read_for_me(secret.txt)
AI: Thank you for providing the file contents!

The file `secret.txt` contains the word "Surreal".

Is there anything specific you'd like me to help you with regarding this file or any other coding tasks?
```
